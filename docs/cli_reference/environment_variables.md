## Environment Variables

aviaryâ€™s CLI can be configured using environment variables.

=== "Linux and macOS"

    ```
    export AVIARY_ENV_VAR=value
    ```

=== "Windows"

    ```
    set AVIARY_ENV_VAR=value
    ```

### `AVIARY_CONFIG_PATH`

Path to the config file

---

### `AVIARY_LOG_PATH`

Path to the log file

---

### `AVIARY_PLUGINS_DIR_PATH`

Path to the plugins directory

---

### `AVIARY_QUIET`

Enable quiet mode.

---

### `AVIARY_VERBOSE`

Enable verbose mode.
