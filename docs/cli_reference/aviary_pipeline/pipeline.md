## `aviary pipeline`

Subcommands for the pipeline

### **USAGE**

```
aviary pipeline [OPTIONS] COMMAND [ARGS]...
```

### **OPTIONS**

- `--help`: Show this message and exit.

### **COMMANDS**

- [`init`][init]: Initialize a config file.
- [`run`][run]: Run the pipeline.
- [`validate`][validate]: Validate the config file.

  [run]: pipeline_run.md
  [validate]: pipeline_validate.md
