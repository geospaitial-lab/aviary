    • Aviary

    aviary provides a command-line interface (CLI) that allows you to run the pre-built pipelines easily without writing any code, requiring only a declarative configuration file that specifies the components.

    To see the available options and commands of the CLI, run the following command:

    aviary

    docker run --rm aviary

    You can choose from the following pipelines:

    • Segmentation pipeline
      Run a segmentation model on your data

    • Postprocessing pipeline
      Postprocess geospatial data
